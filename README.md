# UniPlanner Web Application

## Project Description
UniPlanner is a web-based application designed to help students manage assignments, tasks, and deadlines across multiple courses. It provides a unified calendar view with color-coded priorities and progress tracking.

## Technologies Used
- **Backend:** Java Servlets, JDBC
- **Database:** HSQLDB
- **Server:** Apache Tomcat 9
- **Frontend:** HTML, CSS
- **Version Control:** Git/GitHub

## Features
- User Registration and Login
- Add, Edit, Delete Tasks
- Calendar View with Color Coding (Red = To Do, Yellow = In Progress, Green = Completed)
- Priority Levels (High, Medium, Low)
- Progress Tracking (Kanban-style: To Do / In Progress / Completed)

## How to Run the Application
1. Clone this repository
2. Import the project into Eclipse (File → Import → Existing Projects into Workspace)
3. Configure Tomcat 10.1 server
4. Add `hsqldb.jar` to the project's classpath
5. Deploy and run on Tomcat 10.1
6. Access the application at: `http://localhost:8080/UniPlannerWebApp`

## Database Setup
The application uses HSQLDB. The database is automatically created when the application runs.
